  • watchNodeNotifications(server: Server): Disposable

    Watch server OSC receive for any n_XXX messages:

    • n_go
    • n_end
    • n_on
    • n_off
    • n_move
    • n_info

    Save all of the supplied info for the node and call any registered callbacks.
