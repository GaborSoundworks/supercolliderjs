Class SclangIO

This parses the stdout of sclang and detects changes of the interpreter state and converts compilation errors into js objects.

Also detects runtime errors and results posted when sc code is evaluated from supercollider.js

Convert errors and responses into JavaScript objects

Emit events when state changes
